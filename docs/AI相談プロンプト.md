# 他AIへの相談プロンプト

以下のプロンプトをそのままChatGPT、Gemini、Perplexity等に貼り付けて使用してください。

---

## プロンプト（そのままコピペ可能）

```
あなたはスタートアップのアドバイザーです。以下の企業情報とロードマップを読んで、3つの質問に回答してください。

## 企業情報

- 会社名: 合同会社Reminis（2025年設立、東京）
- プロダクト: FujiTrace（https://www.fujitrace.jp）
- ミッション: 「すべての日本企業にAIを届ける」
- ポジション: AI導入プラットフォーム（AI導入の障壁を取り除く）
- フェーズ: Phase 0（MVP、創業者1人、売上ゼロ、ユーザー数名）
- 法人形態: 合同会社（KK化予定、VC調達準備中）

## プロダクト概要

FujiTraceは、日本企業がAIを安心して導入・運用するためのプラットフォームです。

主な機能:
1. LLMトレース監視 — AIアプリの全リクエストを記録・分析
2. ハルシネーション自動検知 — AIの誤情報をリアルタイムで検出
3. 日本語PII検知 — マイナンバー等15種類以上の個人情報を自動ブロック
4. APIコスト可視化 — ドル円リアルタイム変換で月末の請求ショックを防止
5. AIチャットボット — 月額0円、API従量課金のみ（月1,000会話で約100円）

競合との違い:
- Langfuse（ベルリン発、日本パートナー1社あり）: 開発者向けOSSツール。日本語PII非対応
- LangSmith: 米国、評価額$1.1B。日本市場未進出
- FujiTraceの勝ち筋: 日本語PII検知、国内データ保持、日本の商習慣対応、圧倒的に低いチャットボット価格

## ビジネスモデル

- 無料プラン: 日次30トレース（個人開発者向け）
- Pro: ¥9,800/月（小規模チーム）
- Enterprise: ¥25,000〜¥200,000/月（年間契約）
- チャットボット: プラットフォーム利用料0円 + LLM API従量課金のみ

## GTM戦略

階段型アプローチ:
個人開発者（無料）→ フリーランス（Pro）→ 小規模法人（Enterprise）→ 紹介で拡大

現在の施策:
- Google Ads: チャットボット広告（月2-5万円で需要検証）
- コンテンツ: Zenn/Qiita技術記事
- プレスリリース: 無料配信サービス経由
- X（Twitter）: 個人開発者向け発信

## ロードマップ

Phase 0（現在〜2026年4月末）: 需要検証
- Google Ads チャットボット広告で需要を検証
- プレスリリース配信
- 目標: 有料ユーザー1人 or アクティブ無料ユーザー10人

Phase 1（2026年5月〜7月）: 初期成長
- GK→KK転換（VC調達の前提）
- コンテンツマーケティング強化（月2本）
- チャットボット専用LP作成
- ピッチコンテスト応募
- 目標: ARR ¥100万が見える距離

Phase 2（2026年8月〜10月）: VC調達
- Pre-seed ¥3,000万〜5,000万
- OSSコア公開（GitHub Star獲得）
- 目標: 導入企業10社

Phase 3（2026年11月〜2027年3月）: 法人展開
- Enterprise機能（RBAC、SSO、監査ログ）
- 自律型AIエージェント対応
- 目標: ARR ¥500万、Enterprise顧客1社

## 市場コンテキスト

- 日本のAI導入率は諸外国に比べ大幅に遅れている
- 主因は「技術がない」ではなく「安心して使えない」（信頼性の課題）
- LLMオブザーバビリティ海外市場には$600M以上の投資が集中
- 日本国内には国産競合がゼロ
- Langfuseが日本パートナー（GAO社）経由で進出開始

## 質問

1. **方向性の検証**: このロードマップの方向性は合っていますか？ミッション「すべての日本企業にAIを届ける」を達成するために、最も重要なのに抜けているアクションはありますか？

2. **優先順位**: Phase 0の限られたリソース（創業者1人、広告予算月2-5万円）で、最もインパクトのある行動は何ですか？現在の施策（Google Ads、Zenn記事、プレスリリース、X発信）の優先順位を付けてください。

3. **代替戦略**: 同じミッションを達成するために、このロードマップより効果的なアプローチがあれば提案してください。特に以下の観点で:
   - もっと早く最初の有料ユーザーを獲得する方法
   - 創業者1人でもスケールする仕組みの作り方
   - 日本のSaaS市場特有の勝ちパターン（SmartHR、freee、マネーフォワード等の初期戦略を参考に）

回答は具体的かつ実行可能なアクションで返してください。抽象的なアドバイスは不要です。
```
