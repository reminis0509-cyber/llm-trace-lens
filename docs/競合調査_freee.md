## 1. 調査サマリ

- **調査対象プロダクト数:** 51プロダクト（公開ページ掲載分、8カテゴリ）
- **第2弾候補として ⭐⭐⭐ 評価:** 3プロダクト（AI請求書チェック、AI発注/在庫提案、AI経費精算チェック）
- **法的リスクで ❌ 除外:** 6プロダクト（申告・登記・許認可・就業規則・会社設立・人事労務＝士業独占業務に直接抵触）
- **条件付きOK（要法務確認）:** 契約書レビュー系（freeeサイン契約チェック競合）

### 主要な発見

1. **freeeは「記録・手続き」の自動化に特化しているが、「判断・検証」はほぼ空白。** 請求書を発行する機能はあるが、「受け取った請求書の内容が妥当か」を判断する責任AI領域は freee支出管理 受取請求書 が電子保存レベルに留まっており、AI判定のコア機能は薄い。
2. **freeeの収益構造は月額固定（¥5,000〜¥15,000+ID課金）で、API実費のみモデルとの価格差倍率は最大50〜100倍。** 構造的優位性（Section 7.6）を発揮できる領域が広い。
3. **freeeサイン契約チェック（¥5,000/月、レビュー無制限、弁護士監修）は既に法務省2023年8月ガイドラインの「適法レール」上で稼働している。** FujiTrace が後発で真正面から勝ちにいくより、「士業独占業務に抵触しない周辺領域」を狙うべき。
4. **freeeは AIシフト管理・AIヘルプデスク・AIサーベイなど「AI」を冠したプロダクトを既にリリース済み。** ただし責任の閾値を超える領域（金銭・法的・業務停止）では、AI見積書・AI請求書チェック・AI発注提案がまだ手薄。
5. **申告・登記・許認可など士業独占業務は、freeeが「書類作成ソフト」として既にグレーゾーンを突破している。** FujiTrace が後追いするメリットが薄く、かつ税理士法・行政書士法のリスクが高いため除外推奨。

## 3. プロダクト別詳細分析（高ランク候補のみ）

### 3.1 freee業務委託管理 — ⭐⭐⭐ 候補

**基本情報**
- カテゴリ: 請求販売
- ターゲット: 発注企業（フリーランスと取引する中小企業）
- 公式: https://www.freee.co.jp/products/ 掲載
- 主要機能: フリーランスとの契約・発注・請求・支払の管理、下請法対応
- 料金: 公開情報なし（要問合せ）

**A. 責任の閾値判定**
| 判定 | 理由 |
|-----|------|
| ✅ 金銭 | 下請法違反で勧告・公表・取引停止、支払遅延で遅延利息 |
| ✅ 法的 | 下請法・フリーランス新法（2024施行）違反は行政処分対象 |
| ✅ 信用 | 下請法違反の公表は取引先全体からの信用毀損 |
| ❌ 業務停止 | 直接的な業務停止は稀 |

**B. AI実装可能性**
- 実装難易度: 中
- 必要データ形式: 発注書PDF、契約書テキスト
- Layer 1検証内容: 下請法11項目（支払期日60日以内、下請代金の減額、買いたたき、等）をLLM-as-Judgeで機械的チェック。フリーランス新法の書面交付事項網羅性チェック
- 概算API費用: ¥10〜¥30/件（Claude Sonnet想定）

**C. リーガルチェック**
- 該当法令: 下請法、フリーランス新法、独禁法
- 判定: **条件付きOK** — 「法令適合性の機械的チェック」に留める限り弁護士法72条抵触は低い（AIによる個別具体的な法的判断ではなく、チェックリスト照合）
- 注意点: 「この契約は違法です」と断定せず、「下請法第4条1項に該当する可能性があります。専門家確認推奨」と警告形式で提示する必要あり

**D. 構造的優位性**
- freee価格: 要問合せ（類似カテゴリから月¥10,000〜¥30,000想定）
- FujiTrace想定価格: API実費のみ（月¥300〜¥500）
- 価格差倍率: 約20〜60倍

**E. 推奨度: ⭐⭐⭐**
- 理由: 2024年のフリーランス新法施行で市場が顕在化したばかり。freeeも「管理ソフト」として参入しているが、AI自動チェックは未実装。責任閾値は明確で、法令リスク指摘ツールは警告形式にすれば非弁行為を回避できる。中小企業の発注担当者は専門知識がなく、AIが「下請法違反の可能性」を指摘するだけで強い価値を感じる。

### 3.2 freee支出管理 / 受取請求書チェック領域 — ⭐⭐⭐ 候補

**基本情報**
- カテゴリ: 支出管理
- ターゲット: 経理担当者、中小企業、個人事業主
- 主要機能（freee側）: 受取請求書の電子保存、OCR、債務管理
- 料金: freee支出管理 Fullプラン ¥10,000〜/月相当（推定）

**A. 責任の閾値判定**
| 判定 | 理由 |
|-----|------|
| ✅ 金銭 | 過払い・重複支払・架空請求・インボイス登録番号偽装 |
| ✅ 法的 | インボイス制度（適格請求書の記載事項）違反で仕入税額控除否認 |
| ✅ 信用 | 取引先への誤支払いは関係悪化 |
| ❌ 業務停止 | なし |

**B. AI実装可能性**
- 実装難易度: 中（OCR + LLM判定）
- 必要データ形式: PDF請求書、画像、CSV
- Layer 1検証内容: (1) インボイス登録番号の国税庁APIリアルタイム照合 (2) 金額計算の再検証 (3) 消費税率の整合性 (4) 過去の同一取引先請求との異常値検知 (5) 重複請求検知
- 概算API費用: ¥5〜¥15/件

**C. リーガルチェック**
- 該当法令: 電子帳簿保存法、インボイス制度、消費税法
- 判定: **やってOK** — 税理士法の「税務相談」には該当しない（計算の正誤指摘は税務判断ではない）。国税庁API照合は公開API。
- 注意点: 「この請求書は仕入税額控除できません」と断定せず、「登録番号が国税庁DBで見つかりません」という事実提示に留める

**D. 構造的優位性**
- freee価格: ¥10,000〜¥15,000/月
- FujiTrace想定価格: API実費のみ（月¥200〜¥500）
- 価格差倍率: 約30〜75倍

**E. 推奨度: ⭐⭐⭐**
- 理由: (1) インボイス制度の混乱が継続中で検索需要が高い (2) 個人事業主〜中小企業の「経理は自分でやる」層が顕在ニーズを持っている (3) freee支出管理は「記録・保存」寄りで「チェック・判定」は薄い (4) 法的判断ではなく機械的照合に振れるため法務リスクが低い (5) AI見積書ツール（第1弾）と同じ「作成＆チェック」パラダイムに載る

**D. 構造的優位性**

### 3.5 freeeサイン契約チェック（正面競合） — 参考分析

**基本情報**
- 料金: 月¥5,000、レビュー回数無制限、弁護士監修
- AI機能: NDAや業務委託契約書のAIリスク検知・修正提案

**C. リーガルチェック（最重要）**
- 該当法令: 弁護士法第72条（非弁行為禁止）
- 判定: **条件付きOK（ただし真正面勝負は推奨しない）**
- 根拠: 法務省2023年8月ガイドライン「AI等を用いた契約書等関連業務支援サービスの提供と弁護士法第72条との関係について」により、登録ひな形との差分提示や機械的チェックは合法。ただし「個別具体的な法的判断」に踏み込むと非弁行為の可能性
- freee は弁護士監修を謳うことで事実上の法務フォロー体制を担保している

**E. 推奨度: ❌（正面勝負）／⭐（周辺領域）**
- 理由: freeeサイン契約チェックが既に ¥5,000/月レビュー無制限という強力な価格で先行しており、弁護士監修ブランドもある。真正面で競合するにはFujiTrace側にも弁護士監修体制が必要でスピード感を損なう。ただし「下請法チェック」「フリーランス新法チェック」のような**特定の法律に絞ったチェッカー**なら周辺領域として有望（3.1参照）。

## 4. 戦略的推奨事項

### 4.1 第2弾候補トップ3

1. **AI請求書チェックツール** — インボイス登録番号の国税庁API照合、金額計算再検証、重複・異常検知。Layer 1検証が技術的に最も実装しやすく、法務リスクが最低、ターゲット市場が個人事業主〜中小企業と最大。freee支出管理の月¥10,000超に対してAPI実費¥200〜¥500で構造的非対称性を最大化できる。
2. **AI経費チェックツール** — 領収書OCR + 社内規程照合 + 勘定科目適正性チェック。全法人の普遍ニーズ、freee経費精算の月¥11,000に対する圧倒的価格差、法務リスクほぼゼロ。AI見積書ツール（第1弾）と同じ「作成・入力→自動検証」UXパラダイムでそのまま横展開可能。
3. **AI下請法・フリーランス新法チェッカー** — 発注書・業務委託契約書をアップロードして下請法11項目＋フリーランス新法の書面交付事項を機械的チェック。2024年施行のフリーランス新法で顕在需要が急成長中。freee業務委託管理が「管理ソフト」として先行するもAIチェック機能は未実装。警告形式にすれば弁護士法72条を回避できる。

### 4.2 法的に絶対避けるべき領域

以下は士業独占業務に直接抵触するため、FujiTrace として責任AIツールを出してはならない:

- **確定申告書・法人税申告書の作成補助** — 税理士法第52条（税理士業務の制限）
- **社会保険・労働保険の書類作成・提出代行** — 社労士法第27条
- **就業規則・36協定の作成代行** — 社労士法第27条
- **会社設立登記・商業登記の書類作成** — 司法書士法第73条、行政書士法第1条の2
- **許認可申請書類の作成** — 行政書士法第1条の2
- **具体的な契約書の法的助言・修正提案** — 弁護士法第72条
- **投資助言・個別銘柄推奨** — 金商法第29条
- **診断行為を伴う医療判断** — 医師法第17条
- **マイナンバー取扱い代行** — 個人情報保護法 + マイナンバー法

### 4.3 freee の弱点（隙間）の総括

| 軸 | freeeが強い領域 | freeeが弱い領域（我々の機会） |
|---|---|---|
| **記録 vs 判断** | 会計・勤怠・請求書の記録・保管 | 「記録された情報が妥当か」の判断 |
| **手続き vs 監査** | 申告・登記・契約の手続き自動化 | 「実行した手続きに漏れ/誤りがないか」の監査 |
| **構造化 vs 非構造化** | 定型フォーマットのデータ処理 | PDF/画像/自由記述からの異常検知 |
| **内部整合性 vs 外部照合** | freee内システム連携 | 国税庁API/法令DB等との外部照合 |
| **月額固定 vs 従量** | ¥5,000〜¥15,000/月ID課金 | API実費のみ（¥100〜¥500/月） |

freeeが「記録系」「手続き系」「構造化データ」「月額固定」の4軸に最適化されているため、FujiTrace は「判断系」「監査系」「非構造化データ」「従量課金」の4軸に寄せることで構造的に競合しない非対称ポジションを取れる。

### 4.4 FujiTrace 第2弾の具体実装提案

**プロダクト名案:** FujiTrace AI Tools 第2弾「AI請求書チェッカー」

**想定機能:**
1. PDF/画像/CSVで請求書アップロード
2. OCR＋LLMで項目抽出（請求元、登録番号、金額、税率、品目、支払期日）
3. 【Layer 1 自動検証】
   - 国税庁インボイス登録番号公表サイトAPIでリアルタイム照合
   - 内税/外税と税率の整合性再計算
   - 数量×単価＝金額の再計算
   - 過去30日の同一取引先請求との異常値検知
   - 支払期日の下請法適合性チェック（発注企業モード）
4. 警告表示UIで「ミス検知」体験を提供 → FujiTrace Pro 転換ポイント
5. Free tier: 月5件、Pro tier（¥1,980/月 FujiTrace基本）で月100件、超過時は従量

**想定MVP実装期間:** 2週間（AI見積書ツールのUI/バックエンドを流用可能）

**想定リーガル対応:**
- 「税務相談ではなく機械的照合」という位置付けを利用規約で明示
- 「最終判断は税理士・経理担当にご確認ください」の注記を全UIに配置
- 国税庁APIの利用規約遵守（レート制限、キャッシュ方針）
