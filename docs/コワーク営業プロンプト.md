## 1. ミッション概要

FujiTrace（https://www.fujitrace.jp）の顧客を獲得するため、日本の中小企業にフォーム営業を行う。

**FujiTraceとは:**
- LLM（大規模言語モデル）の可観測性プラットフォーム
- AIチャットbotの品質監視、ハルシネーション検知、コスト追跡を提供
- 日本語PII（個人情報）の自動検出・ブロック機能あり
- 無料プランあり、登録するだけでPro機能を30日間無料体験可能
- https://www.fujitrace.jp

**今回のゴール:**
A/Bテスト用に2パターンのターゲットリストを作成し、各30社にフォーム営業を送信する。

---

## 2. A/Bテスト設計

### パターンA: AIチャットbot搭載済みの企業（30社）
- 既にHP/LPにAIチャットbotを導入している
- 「品質管理ができていない」課題に訴求する

### パターンB: AIチャットbot未搭載の企業（30社）
- HP/LPを持っているがチャットbotがない
- 「チャットbot導入 + 品質管理をセット」で訴求する

---

## 3. ターゲット条件

両パターン共通:

| 条件 | 詳細 |
|------|------|
| 所在地 | 日本 |
| 従業員数 | 50人以下（目安。スタートアップ〜中小企業） |
| 業種 | IT/SaaS、EC、人材、不動産、教育、医療など |
| 必須 | 自社HPに問い合わせフォームがある |
| 除外 | 大企業（資本金50億円以上、従業員1000人以上） |
| 除外 | 競合サービス（Langfuse、LangSmith等のLLM可観測性ツール） |

## 6. フォーム送信手順

### 送信前の準備
1. 上記テンプレートの `[企業名]` `[創業者名]` `[メールアドレス]` を実際の値に置換する
2. フォームの必須項目を確認する（名前、メール、会社名、問い合わせ内容 等）

### 送信フロー（1社あたり）
1. スプレッドシートからフォームURLを開く
2. フォームの各項目に以下を入力:
   - 名前: [村上]
   - メール: [info@fujitrace.jp]
   - 会社名: 合同会社Reminis
   - 電話番号: （フォームに必須の場合のみ入力） 07084632032
   - 問い合わせ種別: 「サービスについて」「その他」等から最適なものを選択
   - 本文: 上記テンプレートの本文部分（件名は除く）
3. **送信ボタンを押す前に、入力内容のスクリーンショットを撮影して確認を取る**
4. 確認OKが出たら送信
5. スプレッドシートの「送信ステータス」列を「送信済み」に更新
6. 「送信日」列に日付を記入

### CAPTCHAがある場合
- フォームの入力までをCoworkが行う
- CAPTCHAの認証と最終送信は創業者が手動で行う
- 備考欄に「CAPTCHA手動送信」と記録する

### 送信ペース
- 1日最大15-20社（スパム判定を避けるため）
- 同一IPから短時間に大量送信しない
- 2-3日に分けて送信する

## 9. 注意事項

- 送信前に必ず創業者の確認を取ること
- 個人情報（電話番号等）は必要最小限のみ入力
- 同じ企業に2回以上送信しない

- フォームの利用規約に「営業禁止」と明記されている場合はスキップ
- 返信が来た企業はすぐに創業者に報告
