## 重要な発見

### 1. 意外な検索量(穴場)
- **「人手不足 倒産」(5,280)** が予想外に大きい。**恐怖訴求は実需が裏づく**。LP Hero やオウンドメディア記事タイトルで強く使える。
- **「請求書 作成 無料」(1,520)** が「請求書 AI」(32)の約 50 倍。**Founder/CEO の言語(AI)とユーザーの言語(無料)に大きなギャップ**。LP の SEO 記事は「無料」を必ず併記すべき。
- **「AI 秘書」(1,280)** はテーマ D 内で突出。**「AI 事務員」より先にユーザーが知っている言葉**。広告クリエイティブで "AI 秘書のように使える" と橋渡し必須。

### 2. 競合の薄い穴場
- **「経理 自動化」(1,920)** はサジェストに「ツール」「ai」「事例」が並び、購入検討段階のユーザーが多い。一方で記事タイトル競合は freee / マネーフォワードの大手中心 → **中小企業特化 + 月¥3,000 という安さで差別化可能**。
- **「中小企業 DX」(1,280)** は補助金狙いの大手 SI が多いが、「**補助金なしで月¥3,000 から始める DX**」という逆張りで差別化可能。

### 3. 避けるべき / 効率の悪いワード
- **「採用 諦めた」「経理 一人しかいない」「人が足りない 経営」** は **検索データなし = ほぼ誰も検索していない**。Founder の体感言語ではあるが、SEO 記事タイトルや広告メインコピーでは使わない方がよい(本文中の感情訴求としては OK)。
- **「請求書 AI」(32)/「見積書 AI」(16)/「AI 事務員」(データなし)** は単独では Google 検索広告に向かない(検索量不足)。**LINE Ads のクリエイティブ語彙、または商品名想起としては OK**。
- **「採用 コスト」(472)** はサジェスト「一人当たり 中小企業」「内訳」が並び、CPC が高い領域の可能性大(コンサル系の流入)。Phase 0 の Google 検索広告では避け、SEO 記事側で長期的に取りに行く方が安全。

### 第2弾の重要発見

#### 1. 「請求書 テンプレート 無料」(5,280) が第1弾「請求書 作成 無料」(1,520) の 3.5 倍

**第2弾最大の発掘**。第1弾推奨で /seikyusho-ai LP の主軸ワードに置いた「請求書 作成 無料」(1,520)よりも、「請求書 テンプレート 無料」(5,280)の方が**圧倒的に大きい**。
ユーザーは「作成ツール」を探しているのではなく、**「Word/Excel に貼れるテンプレート(=書式雛形)」を探している**。
「請求書 エクセル テンプレート」(1,040)も加えると、書類業務テーマは「無料 + テンプレート + エクセル」の 3 語で実需の大半をカバーできる。
**Phase 0 推奨ワード差し替え必須**。

#### 2. 「AI 業務効率化」(14,480) が AI ポジション語の最大ボリューム

第1弾の「AI 秘書」(1,280)を 11 倍上回る。検索者は「秘書(個人秘書のメタファー)」より「業務効率化(B2B ソリューション)」で検索している。ただし**サジェスト「事例/ツール/セミナー」から判断するに、検索意図は B2B 法人・SI 系コンサル流入の可能性が高い**。
- LP Hero での採用は CPC 高騰リスクあり、見送り推奨
- **オウンドメディア記事(SEO)では主軸ワードとして採用**価値あり: 「中小企業向け AI 業務効率化 事例 5 選」など
- LINE Ads クリエイティブでも、Founder/CEO の「AI 事務員」より「AI 業務効率化」のほうが伝わる可能性
