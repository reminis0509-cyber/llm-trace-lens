## 作成ルール
以下の原則に従って書類を作成してください:
1. 日本のビジネス慣習に準拠した書式・敬語を使用すること
2. 宛先・差出人・日付・件名など基本記載事項を漏らさないこと
3. 金額が含まれる場合は算術チェックを行い、小計・税額・合計が整合していることを確認すること
4. 日付は和暦・西暦のいずれかに統一すること
5. 不確実な情報は「[要確認]」と明記し、断定しないこと

{caution_note}
