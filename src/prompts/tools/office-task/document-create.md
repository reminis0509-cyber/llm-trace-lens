## 作成ルール
以下の原則に従って書類を作成してください:

### 基本原則
1. 日本のビジネス慣習に準拠した書式・敬語を使用すること
2. 宛先・差出人・日付・件名など基本記載事項を漏らさないこと
3. 金額が含まれる場合は算術チェックを行い、小計・税額・合計が整合していることを確認すること
4. 日付は西暦表記（例: 2026年4月11日）を基本とすること
5. 不確実な情報は「[要確認]」と明記し、断定しないこと

### 書類種別ごとの必須記載事項
- **見積書**: 見積番号、発行日、有効期限、宛先、差出人、明細（品名・数量・単価・金額）、小計、消費税、合計、支払条件、備考
- **請求書**: 請求番号、発行日、支払期限、宛先、差出人、適格請求書発行事業者番号（インボイス番号）、明細、小計、消費税（税率別）、合計、振込先口座
- **納品書**: 納品番号、納品日、宛先、差出人、明細、備考
- **発注書**: 発注番号、発注日、納期、宛先、差出人、明細、合計、支払条件
- **送付状**: 日付、宛先、差出人、件名、同封物リスト、本文
- **議事録**: 会議名、日時、場所、出席者、議題、決定事項、宿題事項（担当・期限付き）
- **稟議書**: 件名、申請日、申請者、決裁者、目的、内容、費用、効果、リスク
- **報告書**: 件名、報告日、報告者、宛先、要旨、本文、添付資料リスト

### 金額計算ルール
- 消費税は**税率10%**を標準とする（軽減税率8%の品目がある場合は分けて計算）
- 端数処理は**切り捨て**を標準とする
- 合計 = 小計 + 消費税 を必ず検算し、一致しない場合はエラーとして報告すること
