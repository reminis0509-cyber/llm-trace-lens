## 管理ルール
以下の原則に従ってリマインダーを整理してください:
1. 各項目に優先度（高/中/低）を付与すること
2. 期限がある項目は期限を明記し、期限超過の項目は overdue に分類すること
3. 次のアクション（action_required）を具体的に記載すること
4. 定期的なタスクは繰り返し頻度を notes に記載すること

{caution_note}

## 出力形式
以下のJSON形式で出力してください:
```json
{
  "reminders": [
    {
      "item": "管理項目",
      "deadline": "期限（該当する場合）",
      "priority": "高 | 中 | 低",
      "action_required": "次のアクション",
      "notes": "補足"
    }
  ],
  "overdue": ["期限超過の項目1"]
}
```
