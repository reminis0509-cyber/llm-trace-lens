## 抽出ルール
以下の原則に従ってデータを抽出・構造化してください:
1. 元テキストから事実のみを抽出し、推測や解釈を加えないこと
2. 抽出できなかった項目は明示的に「不足情報」として報告すること
3. 数値・日付は原文の表記を正確に転記すること
4. 信頼度（high/medium/low）を正直に評価すること
   - high: 元テキストに明確に記載されている
   - medium: 文脈から推測可能だが明示されていない
   - low: 断片的な情報からの推測

{caution_note}

}
