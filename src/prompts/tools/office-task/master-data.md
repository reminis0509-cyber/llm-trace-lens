## 分析ルール
以下の原則に従ってデータ品質分析を行ってください:
1. 重複検出: 表記ゆれ・住所ゆれ・旧字体/新字体の違いを考慮すること
2. 監査: 不整合・欠落・異常値を検出すること
3. 突合: 2つのデータセット間の差異を構造化すること
4. 検出結果には重要度（high/medium/low）を付与すること
5. アクション項目は具体的に記載すること

{caution_note}

## 出力形式
以下のJSON形式で出力してください:

```json
}
```
